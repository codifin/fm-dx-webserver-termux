# Установка FM-DX Webserver Termux (Android) ([English](https://github.com/codifin/fm-dx-webserver-termux/README.MD))
### !!!РАБОТАЕТ ТОЛЬКО ПОДКЛЮЧЕНИЕ ЧЕРЕЗ WI-FI ИСПОЛЬЗУЯ XDRD!!!

1. Установите Termux через [Google Play](https://play.google.com/store/apps/details?id=com.termux) или [F-Droid](https://f-droid.org/packages/com.termux/)

2. Обнови и установи необходимые пакеты:
```bash
pkg update && pkg upgrade -y
pkg install nodejs-lts ndk-sysroot clang cmake make binutils python ffmpeg git -y
```
3. Склонируйте репозиторий (или скачайте его самостоятельно):

```bash
git clone https://github.com/codifin/fm-dx-webserver-termux/
cd fm-dx-webserver-termux
chmod +x setup.sh
```

4. Запусти скрипт для установки.
```bash
./setup.sh
```

5. Запусти сервер:
```bash
node .
```
Открой интернет-браузер и перейди на http:/localhost:8080 для первичной настройки и дальнейшего использования.

## 🔗 Сопряжение приёмника с телефоном

Для подключения приёмника на базе TEF к телефону, придерживайтесь этих пунктов:

1.  **Включите точку доступа по Wi-Fi**: 
    Включите **Точку доступа** на телефоне на базе Android. Убедись, что тебе известны SSID (название) и пароль созданной сети.

2.  **Подключите приёмник к сети**:
    Включите ваш TEF и подключите его к созданной сети. 
    * *Совет: Проверьте список подключенных устройств для того, чтобы узнать IP приёмника или проверьте его в настройках подключения вашего приёмника.*

3.  **Запуск и настройка**:
    Запустите сервер и пройдите первичную настройку. Оставьте порт по умолчанию:
    * **IP адрес**: Введите IP адрес вашего приёмника (`192.168.x.x`).
    * **Порт**: `7373` (По умолчанию).
    * **Пароль**: Введите ваш xdrd пароль.

## 🐛 Сообщение о ошибках

Если вы столкнулись с ошибкой, пожалуйста, создайте Issue (обращение) и предоставьте следующую информацию:

1. **Логи терминала**: Скопируйте весь текст из Termux (от начала команды до текста ошибки).
2. **Информация о системе**: 
   - Версия Android
   - Модель телефона
   - Результат выполнения команд `node -v` и `uname -m`

> **Примечание**: Большинство ошибок установки в Termux связаны с отсутствием инструментов сборки. Перед созданием обращения убедитесь, что вы выполнили команду `pkg install clang cmake make ndk-sysroot`.
